Show floating social share buttons on the side and size you want.

== Description ==

Plugin show social icons on all page sides. You can choose from 4 icon sizes

Supported social networks:

- Facebook

- Twitter

- Google+

- LinkedIn

- RSS feed



== Installation ==

1. Upload `zedna-social-share-buttons` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

`Cna plugin share every page on my website?`

Yes.

`Can I add more social sites?`

It´s better to ask for adding new social networks in support forum.
